C++ Intro

C++ is a language used to create high-performance applications.

It gives programmers a high level of control over system resources and memory and was developed by Bjarne Stroustrup.

C++ has been updated in 2011, 2014, 2017, and 2020.

Syntax

Breaking Down Code To Understand The Syntax

1. #include <iostream>
2. using namespace std;
3.
4. int main () {
5. cout << "Hello World";
6. return 0;
7. }

Explaining Each Line:

  1. This is a Header File Library for input and output. These header files add functionality to C++ programs.
  2. Gives the ability to use names for objects and variables from the standard library.
  3. Blank Line. C++ ignores the white space but this makes code easier to read.
  4. This is a function. Any code inside the curly brackets will be executed.
  5. cout (Pronounced "see-out") is an object used to output/print text. This line prints Hello World.
  6. This ends the main function.
  7. This ending curly bracket officially ends the main function.

Do Not Forget The ";"

This ends the statement and an error will occur which means the program will not run.

Comments

Comments are used to explain code and to make it easier to read. They can also be used to test alternate code without executing. Comments can be single-line or multi-lined.

Examples:

  • // This is a single-lined comment
  • cout << "Hello World"; // This is a single-lined comment on the same line as code
  • /* This is multi-lined code
    this can take up many lines and comment out entire sections of code */
Variables

Variables are containers for soring data values. There are several different types of variables:

  • int - Stores integers (Whole Numbers)
  • double - Stores floating point numbers (Decimals)
  • char - Stores single characters (a)
  • string - Stores text (Hello World)
  • bool - Stores values with two states (True or False)

Syntax:

type variableName = value;

For Example:

  • int myNum = 15;
  • string myText = "Hello";
Output

The cout object, with the << operator, is upsed to output values/print them.

Example:

cout << "Hello World";
User Input

To get user input, we use cin. This predefined variable reads data from the keyboard with the extraction opperator (>>).

Example:

int x;
cout << "Type a number: ";
cin >> x;
cout << "Your number is: " << x;
